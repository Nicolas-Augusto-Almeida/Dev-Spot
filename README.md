## 🎯 Sobre o Projeto
Este projeto foi desenvolvido como parte de um trabalho acadêmico com o objetivo de criar um aplicativo mobile utilizando Expo e React Native.
O app possui duas telas principais e tem como foco a prática de: 

- Estruturação de componentes
- Estilização de interfaces
- Navegação entre telas
- Organização de projeto
## 📲 Funcionalidades:

### 🏠 Tela Home:
- Exibição do logo e nome do app
- Imagem
- Texto de apresentação:
- Botão "Sobre mim" (navega para a segunda tela)
- Botão "Entre em contato!" (exibe um alerta)
- Seção de redes sociais com redirecionamento para as mesmas
- Switch interativo para mudar entre exibição dark/light
- Rodapé com créditos
### 👤 Tela Sobre Mim:
- Título "Sobre mim"
- Texto descritivo (biografia)
- Layout organizado
- Rodapé com créditos
### 🧭 Tecnologias Utilizadas:
- React Native
- Expo
- JavaScript
### 📁 Estrutura do Projeto:
    /src
      /screens
      /components
      /assets
      /context
  
## 🚀 Como Executar o Projeto:

- Clone o repositório: git clone https://github.com/Nicolas-Augusto-Almeida/Dev-Spot.git
- Acesse a pasta do projeto: cd Dev-Spot
- Instale as dependências: npm install
- Inicie o projeto: npm run start
- Abra no celular com o app Expo Go ou em um emulador
